# Market & Customer Segmentation Analysis

## Overview

This project analyzes customer data using Machine Learning and groups customers into different segments based on their income and spending behavior.

## Technologies Used

* Python
* Pandas
* NumPy
* Scikit-Learn
* Matplotlib
* Seaborn

## Workflow

```text
Dataset
   ↓
Data Preprocessing
   ↓
Feature Scaling
   ↓
K-Means Clustering
   ↓
PCA Visualization
   ↓
Customer Segments
```

## Project Structure
